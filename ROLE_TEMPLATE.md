# Role Template

Use this template to create a new role's skills catalog.

## Steps

1. Create folder: `skills/<role-name>/`
2. Copy this template to: `skills/<role-name>/catalog.md`
3. Fill in the skills table
4. Update `skills/README.md` with new role
